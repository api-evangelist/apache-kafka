# Apache Kafka

> Apache Kafka is an open-source distributed event streaming platform used by thousands of companies for high-performance data pipelines, streaming analytics, data integration, and mission-critical applications. This profile covers its REST-accessible surface: the Kafka REST Proxy API, the Kafka Connect REST API, and the native Kafka messaging protocol (AsyncAPI).

## APIs

- [Kafka REST Proxy API](https://docs.confluent.io/platform/current/kafka-rest/): RESTful interface to a Kafka cluster for producing/consuming messages and managing topics, partitions, consumer groups, ACLs, and cluster state.
- [Kafka Connect REST API](https://kafka.apache.org/documentation/#connect_rest): Manage connectors, configurations, tasks, and offsets for integrating Kafka with external systems.
- [Apache Kafka Messaging API](https://kafka.apache.org/documentation/#producerapi): Core produce/consume protocol over the native Kafka binary protocol with exactly-once semantics and partitioned log storage.
